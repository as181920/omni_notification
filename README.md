通用通知功能(Rails engine)

## Usage

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'omni_notification', git: "git@github.com:as181920/omni_notification.git", branch: "master"
```

And then execute:
```bash
$ bundle
```
